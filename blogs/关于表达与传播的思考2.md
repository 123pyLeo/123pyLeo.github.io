#### 一、核心命题：什么叫「一个可以被解决的问题」

不是“有没有答案”，而是：  
这个问题是否对听众有影响  
这个问题是否真的能被解决  
这个问题是否值得被点开  
所有方法，都是围绕这三点展开。

二、判断一：问题是否「对听众有影响」

1. 影响来自哪里？  
（1）个人利益（共鸣层级）  
一级共鸣：生存级  
暴力 / 性 / 金钱  
二级共鸣：社会关系  
亲情 / 友情 / 职场  
三级共鸣：日常生活  
吃 / 逛 / 买  
共鸣层级越高，点击与投入越强。  
（2）角色 + 情境 + 挑战 = 代入感  
不要只问：  
这个问题是什么？  
要问：  
是谁，在什么情况下，被这个问题困住了？  
换角色、换身份，本质是在扩大共鸣人群。

三、判断二：问题是否「能被解决」

2. 什么才算“答案”  
答案不是宏大理论，而只分三类：  
新事实  
数据 / 现象 / 趋势  
新观念  
经验 / 认知转变 / 感悟  
新技巧  
方法 / 操作 / 技巧 / 流程  
你不是没答案，是没意识到这些本来就是答案。

3. 两种“解决模式”的本质区别  
模式	特点  
医生模式	问题固定，重点找答案  
推销员模式	答案固定，重点找问题  
内容创作 / 表达 ≠ 解题  
而是：  
先有你能给的答案，再去“画靶”。

四、最关键的方法论：先射箭，再画靶

4. 为什么“先射箭”是对的？  
因为：  
生活中到处都是可分享的答案  
但好问题是被设计出来的  
正确顺序是：  
我能给出什么答案？  
这个答案解决的真实困扰是什么？  
谁会被这个困扰击中？  
在什么情境下最痛？

五、让问题「更有吸引力」的两大方法

方法一：换角色 / 换视角（扩大共鸣）  
同一个答案，问给不同人：  
男性护肤  
如何解决男性护肤问题  
如何让老公看起来更有精神  
如何让父亲显得不那么苍老  
答案不变，问题重写。  
方法二：换场景（制造具体冲突）  
不要默认“日常场景”，要找高张力场景：  
加班后见客户  
通宵后第二天出门  
家庭冲突 / 社交暴露  
场景越具体，代入越强。

六、把问题「放大 / 抽象」的技巧

这是非常重要的一步。  
从「操作问题」→「关系问题 / 情绪问题」  
例子：  
如何清理厨房油渍  
如何让男人愿意下厨房  
如何让妈妈做完饭不再暴躁  
如何让孩子对做饭产生兴趣  
从“工具问题”升级为“生活问题”
